# Správa financí / Financial Management

## CZ
Tato aplikace slouží pro jednoduchou správu osobních financí.  
Uživatel si může evidovat své příjmy a výdaje a organizovat je pomocí vlastních kategorií, které patří vždy konkrétnímu uživateli.  
Kategorie si může každý uživatel vytvořit, upravit nebo deaktivovat podle svých potřeb.  
Zadaná data jsou následně přehledně zobrazena ve statistikách, které pomáhají lépe porozumět finančním tokům.  
Cílem aplikace je mít jednoduchý a přehledný nástroj pro kontrolu osobních financí.

## ENG
This application is designed for simple personal finance management.  
Users can record their incomes and expenses and organize them using their own custom categories that belong only to the specific user.  
Each user can create, edit, or deactivate categories according to their needs.  
The entered data are then displayed in clear statistics that help users better understand their financial flows.  
The goal of the application is to provide a simple and clear tool for managing personal finances.
